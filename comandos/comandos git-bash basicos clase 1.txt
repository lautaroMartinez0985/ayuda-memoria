comandos más comunes:

pwd : el comando pwd (print working directory) - impresión del directorio actual.

ls : el comando ls (list) lista el contenido de un directorio

cd : el comando cd (change directory) - cambiar directorio, permite moverse entre directorios del sistema.

mkdir : el comand mkdir (make directory) crear directorio. tener en cuenta que directorios en el lenguaje se refiere a carpetas.

touch : comando que permite crear archivos, al usar este comando siempre hay que usar una extension: ejemplo (touch "prueba.txt"),(touch "prueba.docx")

rm -r : el comando rm -r (remove) tener cuidado l usarlo ya que borra permanentemente el directorio sin retorno.

help : el comando help (ayuda) escribe help para ver la ayuda disponible, ejemplo PDW --HELP para ejemplos de la salida del comando print working directory.

clear : para limpiar el terminal

Si al usar un comando les aparece (commant not found) es porque probablemente lo estan poniendo en mayúsculas
